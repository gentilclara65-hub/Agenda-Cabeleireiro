# 07 - Roteiro de Apresentacao

## Introducao

"Bom dia, professor. Meu projeto se chama Agenda de Cabeleireiro e foi desenvolvido para controlar agendamentos de atendimento profissional em um salao."

## Tema escolhido

"Eu escolhi o tema de controle de agendamento de atendimento profissional, voltado para cabeleireiro, porque e um sistema simples, util e facil de demonstrar."

## Explicacao do backend

"O backend foi feito com Node.js e Express. Ele cria uma API REST com rotas para cadastrar, listar, editar, excluir, marcar atendimento como realizado e cancelar agendamentos."

## Explicacao do banco

"O banco de dados se chama agenda_cabeleireiro e a tabela principal e Agendamentos. Ela guarda o nome do cliente, telefone, data e hora do agendamento, profissional, data do atendimento, data do cancelamento, servico e observacoes."

## Explicacao do frontend

"O frontend foi feito com HTML, CSS e JavaScript puro. Ele consome a API com fetch e mostra formulario, filtros e tabela de registros."

## Demonstracao de cadastrar agendamento

"Agora vou cadastrar um novo agendamento preenchendo os campos do formulario e clicando em salvar. Depois disso o registro aparece na tabela."

## Demonstracao de listar agendamentos

"Aqui eu consigo visualizar todos os agendamentos cadastrados com o status de cada um."

## Demonstracao de editar

"Ao clicar em editar, o formulario e preenchido automaticamente. Depois de alterar os dados, eu salvo novamente para atualizar o registro."

## Demonstracao de marcar como realizado

"Neste botao eu registro que o atendimento foi realizado. O sistema preenche a data de atendimento e muda o status para realizado."

## Demonstracao de cancelar

"Neste outro botao eu registro o cancelamento. O sistema preenche a data de cancelamento e muda o status para cancelado."

## Demonstracao de excluir

"Se eu quiser remover um registro, basta clicar em excluir e confirmar a operacao."

## Encerramento

"Com isso, o projeto atende ao CRUD completo solicitado na atividade, usa banco MySQL, API REST e frontend em JavaScript puro. Obrigado."
